= Декстоп-приложение на Python

Настоящий документ содержит техническое описание декстоп-приложения
на Python с использованием библиотеки Tkinter

Целью данного технического задания является разработка приложения,
которое помогает пользователям формировать график платежей для погашения кредита.
Структура данных приложения приведена на xref:db/db.adoc[странице].

Программа позволяет:

- Вводить данные о сумме кредита, сроке, процентной ставке, варианте погашения ссудной задолженности
- Автоматическое формирование графика платежей с учетом введенных данных
- Отображение графика платежей в виде таблицы или графика (например, диаграммы)
- Сохранение и загрузка данных о платежах
- Генерация отчетов о графике платежей
